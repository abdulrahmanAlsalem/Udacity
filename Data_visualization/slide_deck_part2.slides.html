Investigation Overview¶

My interest to find what are the main characteristics for the majority duration trips.

Dataset Overview¶

447631 bike trips information; inculdes 15 features.
3 features about the trip: Duration, Start and End of (Time & Date)
4 features about the start station: ID, Name, Latitude, Longitude.
4 features about the end station: ID, Name, Latitude, Longitude.
3 features about the rider: User Type (Subscriber or Customer), Year of Birth, Gender.
one about the bike: ID.
